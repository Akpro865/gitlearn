Git commands :

  1. git init => inits a empty git repository
  2. git status => gives gits current status
  3. git commit --amend => appends to last commit
  4. git log "file code" --stat => status of that commit
  5. git log => logs all commit
  6. git config --global alias.st status => stting shortcut
  7. git reset HEAD~1 = backs last commit to staging area
  8. git clean -fd => deleting file and folder
  9. git revert "id" => delets all change and creates new commit
  10. git tag -a v1.0.1 -m "initial release" => tagging a version
  11. git branch => gives available branches
  12. git checkout "id" => goes to that time period code
  13. git checkout master => bacs to last code
  14. git branch "branchname" => switching branch
  15. git merge icons => merge icon branch to master
